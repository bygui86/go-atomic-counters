
# go-atomic-counters
Golang sample project to test atomic counters

---

## Run
```shell
go run main.go
```

Expected output: every 5 seconds should be printed out a line like `Counts: 50`

## Run examples
```shell
go run ./examples/go-by-example.go

go run ./examples/go-playground.go
```
